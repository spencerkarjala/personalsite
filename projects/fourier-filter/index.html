About Projects

A couple years ago, I switched my home machine from Windows to Arch Linux. In the process of doing so, I also switched from Ableton to Bitwig as my main DAW. However, Bitwig does not have the Max MSP interoperability that Ableton does, so I lost access to a lot of Max devices that I used in my production. One of these is katja's Fourier Filter , which I use often for interesting spectral sound design. Since I had recently worked on my diatonic vocoder project , I decided to just rebuild the effect in JUCE/C++.

The effect has six main parameters:

  • Bands: the number of sinusoid periods that affect the spectrum
  • Position: the phase used to create movement (equivalent at 0% and 100%)
  • Width: how narrow each sinusoid period is that affects the spectrum
  • Offset: adds extra of the 'Position' parameters to the left/right channels
  • Bias: re-distributes the sinusoid bands to be more tightly bunched towards lower/higher frequencies
  • Makeup: optional extra gain in case signal is too quiet after modulation

DEMO

The first video below shows the effect working on a simple saw chord. As the width parameter is increased from zero, the width of the periodic bands goes from 0 (ie. off) to an audible range, and the sound becomes glassy and resonant. It also shows off a fun spectrum visualizer I threw together for this project.
