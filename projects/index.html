About Projects

Grid Pathfinding in Unreal Engine

A lightweight module built for Unreal Engine 5.3 that hooks into the existing navigation system to support click-to-move grid-aligned pathfinding, similar to CRPGs like UnderRail or the original Fallout games.

Fourier Filter VST

A VST port of a Max MSP device that I used often prior to switching from Ableton to Bitwig. Original device created by katja at katjaas.nl.

Diatonic Vocoder VST

An audio effect plugin that, when given a musical scale and a sound, decreases the volume of the parts of that sound that don't fit within the notes of that scale.

CPU Raytracer

A simple CPU-operated raytracing renderer that I built for a class in university, similar to Raytracing in One Weekend.
